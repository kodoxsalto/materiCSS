Penulisan CSS bisa melalui 3 cara

INLINE

teks ini ditulis dengan style css berupa inline dengan menambahkan atribut style pada tag

INTERNAL

Kalau teks ini ditulis dengan style css metode internal dimana pemberian style di tulis di < head > menggunakan tag < style >

Coba diperhatikan padahal jika kita menggunakan style css metode internal maka dia akan mempengaruhi semua selector pada 1 halaman. Namun sekarang pertanyaannya adalah mengapa tag P pada INLINE diatas tidak ikut berubah menjadi warna biru ??

Hal ini karena pada CSS terdapat sebuah hirarki prioritas dimana prioritas pertama adalah metode INLINE, INTERNAL dan terakhir adalah EKSTERNAL

Oleh karena itu jika dilihat dengan paragraf inline sebenarnya memiliki 2 opsi warna teks yaitu biru dan merah. biru didapatkan dari pemberian style secara INTERNAL dan merah pemberian secara INLINE. Namun, yang ditampilkan oleh HTML adalah warna Merah karena ada prioritas tadi

EKSTERNAL

Teks ini walau berada pada subbab penulisan secara EKSTERNAL namun warna teks ini sama seperti teks INTERNAl. Hal ini karena jika kita memberikan style warna teks menggunakan metode eksternal maka tetap saja warna yang akan dimunculkan pada HTML adalah warna biru karena INTERNAL lebih di prioritaskan daripada EKSTERNAL. Oleh karena itu kita akan memberi style pada selector H4 saja
